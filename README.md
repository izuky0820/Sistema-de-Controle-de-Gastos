# Sistema de Controle de Gastos Residenciais

## Objetivo

Aplicação desenvolvida para gerenciar gastos residenciais, permitindo o cadastro de pessoas, cadastro de transações financeiras e consulta dos totais por pessoa e do sistema.

## Tecnologias

- ASP.NET Core (.NET)
- C#
- React
- TypeScript
- Axios

## Persistência

Como o projeto não utiliza banco de dados, os dados são persistidos em um arquivo JSON (`Data/dados.json`).

## Estrutura

Backend

- Controllers
- Models
- DTOs
- Services

Frontend

- Components
- Pages
- Services
- Types

## Funcionalidades

- Cadastro de pessoas
- Remoção de pessoas
- Cadastro de transações
- Consulta de totais
- Persistência dos dados
