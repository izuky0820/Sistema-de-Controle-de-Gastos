# Sistema-de-Controle-de-Gastos


## Tecnologias

- .NET 10
- ASP.NET Core
- C#
- React
- TypeScript

## Persistência

Os dados são armazenados em um arquivo JSON (`Data/dados.json`), sem utilização de banco de dados.

## Funcionalidades

- Cadastro de Pessoas
- Cadastro de Transações
- Consulta de Totais
